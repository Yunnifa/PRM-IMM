# Meeting Room Calendar - Frontend

Aplikasi kalender untuk menampilkan penggunaan ruangan meeting.

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite

## Instalasi

```bash
npm install
```

## Menjalankan Development Server

```bash
npm run dev
```

Aplikasi akan berjalan di http://localhost:3000

## Build untuk Production

```bash
npm run build
```

## Fitur

- ✅ Menampilkan kalender bulanan dengan tanggal real-time
- ✅ Header "Penggunaan Ruangan Meeting"
- ✅ Navigasi bulan (sebelumnya/berikutnya)
- ✅ Highlight hari ini
- ✅ Tampilan full-screen dengan desain modern
- ✅ Responsive design dengan Tailwind CSS

## Struktur Komponen

- `App.tsx` - Komponen utama aplikasi
- `components/Calendar.tsx` - Komponen kalender utama dengan logika dan UI
